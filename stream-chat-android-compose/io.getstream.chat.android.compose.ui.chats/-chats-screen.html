ChatsScreen

@Composable
fun ChatsScreen(modifier: Modifier = Modifier, channelViewModelFactory: ChannelViewModelFactory = ChannelViewModelFactory(), channelViewModelKey: String? = null, title: String = "Stream Chat", isShowingHeader: Boolean = true, searchMode: SearchMode = SearchMode.None, onChannelsHeaderAvatarClick: () -> Unit = {}, onChannelsHeaderActionClick: () -> Unit = {}, onViewChannelInfoAction: (channel: Channel) -> Unit = {}, onMessagesHeaderTitleClick: (channel: Channel) -> Unit = {}, onMessagesHeaderAvatarClick: (user: User) -> Unit = {}, onNavigateToMessages: (channelId: String?, singlePanel: Boolean) -> Unit = { _, _ -> }, onBackPressed: () -> Unit = {})

Default Chat screen component, that provides adaptive layout based on the screen size.

It can be used without most parameters for default behavior, that can be tweaked if necessary.

Parameters

modifier

The modifier to be applied to the layout.

channelViewModelFactory

The factory used to build the ChannelListViewModel and power the behavior. You can use the default implementation by not passing in an instance yourself, or you can customize the behavior using its parameters.

channelViewModelKey

Key to differentiate between instances of ChannelListViewModel.

title

The header title.

isShowingHeader

If a header should be shown.

searchMode

The search mode for the screen.

onChannelsHeaderAvatarClick

Called when the user clicks on the channels header avatar.

onChannelsHeaderActionClick

Called when the user clicks on the channels header action.

onViewChannelInfoAction

Called when the user clicks on the view channel info action.

onMessagesHeaderTitleClick

Called when the user clicks on the messages header title.

onMessagesHeaderAvatarClick

Called when the user clicks on the messages header avatar.

onNavigateToMessages

Called when the user navigates towards a channel or navigates back from a channel. The channelId is null when user navigates back from a channel.

onBackPressed

Handler for back press action.
